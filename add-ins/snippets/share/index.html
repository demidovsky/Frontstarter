Notes:
  1. When location and og:url doesn't match - og:url is the winner.
  2. Facebook goes to current page, then takes url from og tag there, and then scrapes data only from that url.
  3. Google+ scrapes all data from current page, except location – it is taken from og:url tag.
  4. Although the only thing Twitter shows in sharing dialog is url, it is appeared as card then.
  5. Do not forget to clean FB cache
