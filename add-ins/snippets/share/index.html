Шаринг текущей страницы (возможен только на внешнем сервере)

  • image:
  • title: Заголовок страницы
  • description: Описание страницы
  • url (document.location):
  • og url: https://github.com/demidovsky/Frontstarter/
Заметки:
  1. Если переданный адрес и og:url не совпадают — Google+ парсит данные с того, что передали, но ссылка в самой записи будет из og:url.
  2. Остальные отобразят в записи переданный адрес.
  3. Кроме Facebook — он переходит по переданному адресу, но ищет там og:url, и парсит данные уже по нему.
  4. Twitter в окне шаринга показывает только ссылку, но в ленте отображается карточка с данными.
  5. Вконтакте берёт последний из метатегов :title, но первый из метатегов :description — т.е. слегка конфликтует с твиттер-карточками.
  6. Для обновления данных в Facebook нужно сбросить его кэш.
