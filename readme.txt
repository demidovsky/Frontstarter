Frontstarter - это фреймворк для быстрой вёрстки проекта с нуля.
Он помогает организовать файлы, а также быстро реализовывать часто встречающиеся задачи:

- кнопки
- скроллбары
- слайдеры
- формы
- модальные окна
- подсказки
- прилипающий скролл
- полноэкранный скролл
- параллакс
- антикэш
- кастомные шрифты
- заглушка для старых браузеров
- to be continued...


HTML
    /templates - повторяющиеся на разных страницах фрагменты
    /fonts     - сконвертированные для веб шрифты
    demo.php   - примеры типовых решений
    index.html - список страниц нового проекта
    lego.php   - страница с компонентами нового проекта
    empty.php  - пустая страница как основа для создания новых

CSS
    /default             - библиотеки
    /default/snippets    - примеры
    /default/mixins.less - коллекция миксинов для CSS3, а также ряд фишек
    /modules             - общие для всех страниц стили
    /pages               - индивидуальные стили по страницам
    empty.less           - пустой файл как основа для создания новых

JS
    /default          - библиотеки
    /default/snippets - примеры
    /modules          - общие для всех страниц скрипты
    /pages            - индивидуальные скрипты по страницам
    empty.js          - пустой файл как основа для создания новых

IMG
    /pages   - индивидуальные картинки по страницам
    /arrow   - стрелки
    /bg      - фоны
    /content - тестовое наполнение нового проекта
    /icon    - значки, лоадеры и прочая мелочь
    /logo    - логотипы
    /jreject - для старых браузеров
